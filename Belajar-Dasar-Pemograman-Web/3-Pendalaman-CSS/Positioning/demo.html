Demo Jenis Positioning CSS

Contoh static, relative, absolute, fixed, dan sticky digambarkan dengan box.

static relative absolute fixed sticky

1) position: static (default)

Elemen mengikuti alur normal dokumen. Properti seperti top/left tidak berpengaruh.

STATIC

2) position: relative

Elemen tetap menyisakan ruang aslinya, namun dapat digeser relatif terhadap posisi normalnya.

RELATIVE

3) position: absolute

Diposisi absolut terhadap ancestor terdekat yang positioned (bukan static). Jika tidak ada, relatif ke viewport.

ABSOLUTE

4) position: sticky

Berperilaku seperti relative hingga menyentuh ambang top, kemudian menempel di tepi kontainer scroll.

STICKY HEADER (top: 0)

Scroll di area ini untuk melihat efek sticky.
